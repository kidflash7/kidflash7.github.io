Lab11: PNZ Game

PNZ: In the game of PNZ the computer generates a random number consisting of 3 distinct digits ... (ie No digit is the same). The user trys to guess the number and receives the following hints from the computer:
Each P means that a digit is in the correct position without saying saying which position.
Each N means that a digit occurs in the number, but it's not in the correct position.
A single Z means no digits are in the number.

Assume that the magic random number is 123:
GUESS HINT
456 Z
345 N
134 PN
431 NN
143 PP
213 PNN
123 PPP
