Lab 17: PNZ Game

PNZ: In the game of PNZ the computer generates a random number consisting of 3 distinct digits ... (ie No digit is the same). The user trys to guess the number and receives the following hints from the computer:
Each P means that a digit is in the correct position without saying saying which position.
Each N means that a digit occurs in the number, but it's not in the correct position.
A single Z means no digits are in the number.

Assume that the magic random number is 123:

Lab 18: Play BlackJack vs Computer

BlackJack aka 21: Write the Code to play 21 against the computer. The Computer will be the Dealer and you will play against the computer (dealer). Go on-line to get all the information on how the game is played. One of the hardest parts of the code is keeping track of which cards have been dealt, I would simplify things by making an infinite deck Shoe.

Let's break this project down into smaller programs:
Program 1: Write the code to deal cards: [K♥] [7♠] The user clicks a button and you deal a new card.

Program 2: Add the code to Program 1 that will output the value of all the dealt cards. The value of the card is the number on the card + Ace = 11 + Face Cards = 10.

Program 3: Write the code to play BlackJack with the computer with only two cards being dealt to you and dealer

Program 4: Now write the full code to play Blackjack against the computer(dealer).

The objective of the game is to beat the dealer in one of the following ways:
  • Get 21 points on the player's first two cards (called a "blackjack"), without a dealer blackjack
  • Reach a final score higher than the dealer without exceeding 21
  • or Let the dealer draw additional cards until their hand exceeds 21 ("busted")
Note: Dealer and Player are dealt two cards.
Note: Player (you) get's the opportunity to hit (get additional cards) up until you exceed 21 (bust, dealer wins).
Note: An Ace is worth 11 unless your hand is over 21 then the Ace will turn to the value of 1.
Note: Dealer has to hit until they reach 17, then they must stay.
Note: Even score is a push (draw)
