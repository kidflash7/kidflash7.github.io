Arrays: List of variables of the same data type

var arr = [];
var i, rnum;

arr.length = 10;

for (i=0; i<=9; i++)
{
      rnum = Math.ceil(Math.random()*100);
      arr[i] = rnum;
}

The above code first declares an array ( arr = []; ) and then it declares how many elements are in the arrary ( arr.lenght = 10; ) the for loop is used to assign a random number to each array element. Arrays always start at 0 and go up by one. If I want to write the 1st element in the array document.write( arr[0] ); and if I want to write the 10th element in the arrary document.write( arr[9] ); Arrays are a nice way to work with a lot of data
