Lab 3: Fahrenheit to Celsius

Have User input the temperature in Fahrenheit, Convert temperature to Celsius. Output both temperatures and output the sport appropriate for the temperature. You must have at least 4 possible outputs.

When you first write programs you need to start off simple and small, get it to work then add more code a little at a time, test it, get to work, repeat this process until code is complete.

First 4 labs stick with simple input and output too, input using prompt and output using document.write

<script language="javascript">

var fahr;
var sport, cr;

cr = "<br>";
sport = "Bowling";

fahr=prompt("Type in temperature (Fahr) ",70);

if ( ( fahr>=80 ) && ( fahr<=100 ) )
{
      sport = "Swimming";
}

document.write(" Temperature in Fahr = ");
document.write(fahr);
document.write(cr);
document.write(" The sport is ... ");
document.write(sport);
