Loops

Do a series of commands over and over again

for loop:
for ( i=1; i<=10; i++ )
{
    document.write(i);
}
Execute Statement(s) inside loop starting at i=1 and repeating until i=10, incrementing i each time through loop by (1), but you can increment or decrement by anything you want.

i is called a counter, usually you reserve counters to be variable names of i, j, and k.

counters are extremely important in programming: i++ is the same as i=i+1. i=i+1 increments i by 1

I usually use a for loop if I want to increment or decrement by 1

copy and paste above loop inside java script to see how it works

while loop:
i=1;
while ( i<=10 )
{
    document.write(i);
    i = i + 1;
}
j=1;
while ( j<=10 )
{
    document.write(j);
    j = j + 0.1;
}
The left while loop outputs the same thing as the for loop above, but you'll need to initialize the counter before the loop starts (i=1) and increment the counter manually inside the loop (i=i+1)

The right while loop outputs: 1, 1.1, 1.2, 1.3, ... , 9.7, 9.8, 9.9, 10

I usually use a while loop if I want to increment or decrement a counter by something other than 1 say 0.1: i = i - 0.1

do ... while loop:
i=1;
do
{
    document.write(i);
    i = i + 1;
} while ( i<=10 )
This outputs the same thing as the first for loop but once again you need to initialize the counter before the loop and increment the counter manually inside the loop. Also the do while loop will always executes the loop at least once since it doesn't check to kick out of the loop until the end of the loop

continue:
i=0;
while ( i<=10 )
{
    i = i + 1;
    if ( i == 5 )
    {
       continue;
    }
    document.write(i);
}
Skips an iteration of the loop when i=5

break:
i=1;
while ( i<=10 )
{
    if ( i == 5 )
    {
       break;
    }
    document.write(i);
    i = i + 1;
}
Breaks out of the loop when i=5
