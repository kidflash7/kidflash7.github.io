JavaScript Variables
Before you use a variable in a JavaScript program, you must declare it. Variables are declared with the var keyword as follows:

<script type="text/javascript">

var first, last;
var num1, num2;

...

Storing an initial value in a variable is called intializing the variable, it should be done after you declared your variables.

first = "BillyBob";
last = "Applegate";
num1 = 7;
num2 = 10.25;

Note: JavaScript is untyped language: A JavaScript variable can hold a value of any data type. Unlike many other languages, you don't have to tell JavaScript during variable declaration what type of value the variable will hold. The value type of a variable can change during the execution of a program.

Arithmetic & Assignment Operators:

Assignment Operator =
Arithmetic Operators:
+ (Addition)
- (Subtraction)
* (Multiplication)
/ (Division)
% (Modulus - Remainder of an integer division)
++ (Increment variable by 1)
-- (Decrement variable by 1)

Note: Addition operator (+) works for Numeric as well as Strings: "B" + 12; will give "B12".
