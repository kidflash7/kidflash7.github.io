User Defined Functions

function fun1(var1, var2)     // var1, var2 optional
{
      var sum1;                            // local variable

      sum1=var1 + var2;

      return sum1;                        // optional
}

User Defined Functions:
Reuseable code that can be called anywhere in your program, but in order to execute it, it must be called i.e. Math.random(); is a function, all calls to reserved words in JavaScript are calls to functions in the Web Browser

JavaScript should be written as a series of functions inside the <head> tag in HTML. You can call on other Functions from within a Function

All your programs from now on should just be calls to functions:

  1. insert the <script> tag in your <head> tag after the <title> tag
  2. declare all your variables
  3. function to initialize your variables (i.e. sum = 0;)
  4. function to input data
  5. function for calculations
  6. function to output your results
  7. function to call all your functions
