arr1.length = 10;

for (i=0; i<=9; i++)
{

var arr1 = [];
var i, num, length_of_array;

for (i=0; i<=9; i++)
{
      num = i + 100;
      arr1.push(num);
}

length_of_array = arr1.length;

The above code(left) first declares an array ( arr1 = []; ) and then it declares how many elements are in the arrary ( arr1.lenght = 10; ) the for loop is used to store the numbers 100 to 109 in the array. Arrays always start at 0 and go up by one. If I want to write the 1st element in the array document.write( arr1[0] ); and if I want to write the 10th element in the arrary document.write( arr1[9] ); Arrays are a nice way to work with a lot of data
