JavaScript

A programming language insterted right into webpages to make your webpages dynamic(interactive).

    Why JavaScript
  • Relevancy to the GUI generation
  • No compiler needed, the interpreter is built into your web browser
  • Not everyone is gonna get programming, but everyone can get HTML

JavaScript statements
are placed within the <script> ... </script> HTML tags in a webpage.

Seperate JavaScript statements with ;

Enabling JavaScript in your browser:
JavaScript in Internet Explorer
Follow Tools ? Internet Options from the menu. Select Security tab from the dialog box. Click the Custom Level button. Scroll down till you find Scripting option. Select Enable radio button under Active scripting. click OK

JavaScript in Chrome:
Select Settings. Click Show advanced settings at the end of the page. Under the Privacy section, click the Content settings button. In the "Javascript" section, "Allow all sites to run JavaScript".
