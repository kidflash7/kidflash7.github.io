Lab 16: Find the Equation of a Line

Write a program to find the the equation of a line in a plane.
The user will input the coordinates of any 2 unique points in a plane
and you will find the equation of the line that goes through those points
Input: (x1, y1), (x2, y2) Output: y=mx+b or Ax+By=C



Lab 17: Perfect Integers up to 1000

Write a program that finds all perfect integers up to 1000. A perfect integer is
a positive integer which is equal to the sum of all its factors except itself.
For example 6 is a perfect integer: 1 + 2 + 3 = 6
