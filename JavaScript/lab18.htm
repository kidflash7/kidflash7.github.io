Lab 20: Approximate Roots of a 3rd Degree Poly

Approximate the root of a 3rd degree polynomial function that has real roots using the bisection method. User will input coefficents(a, b, c, d) for the 3rd degree polynomial function: ax3 + bx2 + cx + d = 0. The user will also have to input two integers (m, n) such that they know a root and only one root exists between these two x-values. I know a root exists between m and n if f(m)>0 and f(n)<0 or f(m)<0 and f(n)>0. Bisection Method: Let's assume: n>m and f(m)<0 and f(n)>0 a good guess for the root is halfway between m and n: mid1 = (m + n)/2 now if f(mid1)>0 you have a new n but if f(mid1)<0 then you have a new m now test to see if abs(m-n) < error if it isn't repeat this process until abs(m-n) < error then output f(m) or f(n) Note: error is what you decide it to be in your program i.e. error=0.0001 this will produce a root accurate to 3 places past the decimal.
