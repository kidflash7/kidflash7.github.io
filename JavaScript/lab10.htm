Lab 11 - Stats

Generate 20 Random Numbers: 1-100. Find: Mean, Median, and Standard Deviation

Note: The hardest part of the program will be finding the Median, because you will have to sort the numbers.

Note: One of the easiest sorting routines to understand is Selection Sort: Let the first number in the Array (num[0]) be the smallest, then loop through the rest of the numbers and find the smallest number, then swap the two numbers in the array. Repeat the process with the second number in the Array (num[1]) and keep repeating this process up until the next to last number in the array.

  1. Write code to loop through and find the smallest number in the array and it's location
  2. Write code to interchange 1st array element with smallest number in the array
  3. Now you are ready to write the complete sorting code
