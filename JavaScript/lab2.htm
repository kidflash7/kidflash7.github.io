JavaScript: Variables with Arithmetic & Assigment Operators

Lab 2:

Declare 6 variables to store numbers(num1, num2, sum, prod, diff, div) and 3 variables to store strings(first, last, fullname). Perform Arithmetic Operators on the two numbers and output your full name followed by the results of arithmetic operations on the two numbers.

Becareful with Numbers! Since you don't have to declare what type your variables are, sometimes the JavaScript Interpreter treats numbers as strings, which is fine unless you want to preform math calculations. To overcome this you might initially have to make use of the built in function Number(num1) which forces variable num1 to be a number.

Becareful; you CANNOT use reserved words in JavaScript for Variable Names (i.e. break, if, for, while, do, ...)


It is very important to develope good habits when writing code:

  • Plan before coding
  • Write a very simple/basic version first and get it to work (debug)
  • Use Indentation and blank lines for readability
  • Divide program into the following ordered blocks of code
    1. Variable declarations
    2. Initialize variables
    3. Input
    4. Calculations
    5. Output
  • Comment on what your code does (Internal Documentation)
