RAM - Random Access Memory

Primary Memory

Short-term data storage. Stores the information your computer is actively using so that it can be accessed quickly

The CPU uses the RAM to store data needed for running the OS and all programs stored on your computer's HDD

I have 8 GB of DDR3 SDRAM on my computer

Pro's: Can access data fast as compared to secondary memory (HDD)

Con's: Not a lot of storage as compared to HDD. It's volatile: Data is lost when computer is powered down
