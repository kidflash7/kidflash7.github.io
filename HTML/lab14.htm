BackGround Music

Internet Explorer

< bgsound src="tune.MID" loop=n >

n : 1 to infinite (play # times)

Netscape

< embed src="tune.MID" loop=n hidden=true autostart=true >

n : 1 to true (play # times)

Most music used in web pages are MIDI's because they take up less memory. You can use a WAV file for sound bites, but they better be short because they take up a lot of memory. You should have an option for the user to turn off the music. The way to change the music or turn it off without Java is to make a whole new page exactly the same and just change the file in the BGSOUND tag or don't even include it if the user chooses Stop Da Music.
