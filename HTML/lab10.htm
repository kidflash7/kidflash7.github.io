TO SPECIFY THE THICKNESS OF THE BORDER:
< table border=n >

TO SPECIFY THE BORDER COLOR:
< table bordercolor=color >

ADDING CELL SPACING AND CELL PADDING:
< table cellspacing=n cellpadding=n >

CHANGE TABLE SIZE:
< table width=x height=y >

CHANGE SIZE OF EACH CELL:
< th width=x height=y > .. < /th >
< td width=x height=y > .. < /td >

SPAN A CELL ACROSS 2 or MORE COLUMNS:
< th colspan=k > .. < /th >
< td colspan=k > .. < /td >

SPAN A CELL ACROSS 2 or MORE ROWS:
< th rowspan=k > .. < /th >
< td rowspan=k > .. < /td >

ALIGN A CELL's CONTENTS HORIZONTALLY:
< th align=direction > .. < /th >
< td align=direction > .. < /td >

ALIGN A CELL's CONTENTS VERTICALLY:
< th valign=direction > .. < /th >
< td valign=direction > .. < /td >

CHANGE A CELL's COLOR:
< th bgcolor=#ff1000 > .. < /th >
< td bgcolor=#ff1000 > .. < /td >

KEEP TEXT IN A CELL ON ONE SINGLE LINE:
< th nowrap > .. < /th >
< td nowrap > .. < /td >

n : pixels
x,y : % of window or pixels
k : # of rows or columns
direction : left, center, or right
