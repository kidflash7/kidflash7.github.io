3 important things to remember in HTML:
(1) All content goes between beginning <p> ... some text content ... </p> and ending tags
(2) All tags <p> need to be ended </p> , exceptions: <br> and <img src>
(3) All tags need to be nested:

<body>

    <p>
        paragraph tag nested in the body tag
    </p>

</body>
