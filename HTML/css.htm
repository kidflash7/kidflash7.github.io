Intro to Cascade Style Sheets - CSS - Styles

Internal CSS is placed between the <style> tag inbeded inside the <head> tag. A style is made with a selector followed by a beginning set bracket: p{ You then define how the elements marked with that tag should be displayed in your page followed by ending set bracket.

External CSS is placed in seperate text file ("file_name.css") and then called upon inside the <head> tag with the following tag:
<link href="file_name.css" rel="stylesheet" type="text/css">
All the external CSS file contains is the selectors followed by all the properties placed within set brackets, no need for the style tag.
