FRAMES

CREATE A SIMPLE FRAMESET IN ROWS:
< frameset rows="a,b" >
< frame name="first_row" src="row1.html" >
< frame name="sec_row" src="row2.html" >
< /frameset >

CREATE A SIMPLE FRAMESET IN COLUMNS:
< frameset cols="d,e" >
< frame name="first_col" src="col1.html" >
< frame name="sec_col" src="col2.html" >
< /frameset >

CREATE A FRAMESET with 3 ROWS & 2 COLS in 2nd ROW only:
< frameset rows="a,b,c" >
< frame name="first_row" src="row1.html" >
< frameset cols="d,e" >
< frame name="row2col1" src="row2col1.html" >
< frame name="row2col2" src="row2col2.html" >
< /frameset > this ends frameset for columns
< frame name="third_row" src="row3.html" >
< /frameset > this ends frameset for rows

ADJUST A FRAME's MARGINS:
< frameset rows="a,b" >
< frame name="row1" src="row1.htm" marginwidth=w marginheight=h >
< frame name="row2" src="row2.htm" marginwidth=w marginheight=h >
< /frameset >

TO MAKE ALL THE BORDERS DISAPPEAR:
< frameset rows="a,b" frameborder=0 >

TO MAKE ONLY THE VERTICAL BORDERS DISAPPEAR:
< frame name="name" src="contents.htm" frameborder=0 >

TO TARGET A LINK TO A PARTICULAR FRAME:
< a href="contents.htm" target="name_of_frame" >

TO TARGET A LINK TO A SPECIAL SPOT:
< a href="contents.htm" target="_blank" > open in a new tab
< a href="contents.htm" target="_self" > open in frame that contains link
< a href="contents.htm" target="_top" > open in place of existing page
< a href="contents.htm" target="_parent" > open in frame that contains current frameset

NOTE: a,b,c: height of rows d,e: width of columns
height, width: pixels, %, or * (rest of page)

They say in the near future web browsers won't support frames, but until that day comes I will use them to make my big webpages read like a book with chapters. I'm partial to a left vertical navigational bar. This is the only thing frames are good for: letting the side navigational bar stay the same and just put new pages in the main frame.

If you think you want to go into coding you should probably use HTML5 and CSS and figure out a way to make a menu bar without frames.
