CREATE A SIMPLE TABLE WITH HEADERS ACROSS THE TOP:
< table border=5(number of pixels) >
< caption > the caption statement to use with table < /caption >
< tr > (this defines the beginning of the first row)
< th > header for first column < /th >
< th > header for the second column < /th >
(repeat < th > ... < /th > for each column)
< /tr >
< tr > (this defines the beginning of the second row)
< td > put in table data of cell in first column < /td >
(repeat < td > ..... < /td > for each column)
< /tr >
(repeat < tr > ... < /tr > for each row)
< /table >

CREATE A SIMPLE TABLE WITH HEADERS ALONG THE LEFT SIDE:
< table >
< tr > (this defines the beginning of the first row)
< th > header for first row < /th >
< td > table data of cell in first column < /td >
(repeat < td > ... < /td > for each column)
< /tr >
< tr > (this defines the beginning of the second row)
< th > header for the second row < /th >
< td > put in table data of cell in first column second row < /td >
(repeat < td > ..... < /td > for each column)
< /tr >
(repeat < tr > and also < th > ... < /th > for each row)
( < td > ... < /td > for each column)
< /tr >
< /table >

You can also combine both methods and have headers on top and to the left of your data cells. The following is a simple example of a two row three column table with headers at the top of each column:

< table border=5 cellspacing=5 cellpading=4 width=70% bordercolor=black >
< tr align=center >
< th > East Coast < /th >
< th > West Coast < /th >
< th > No Coast < /th >
< /tr >
< tr align=center >
< td > 200 < /td >
< td > 100 < /td >
< td > 10 < /td >
< /tr >
< tr align=center >
< td > 102 < /td >
< td > 205 < /td >
< td > 3 < /td >
< /tr >
< /table >
