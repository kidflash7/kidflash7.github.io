TEXT FORMATTING GUIDE:
<br> carriage return tag, one of the few tags that doesn't need to be ended
&nbsp;     inserts a blank space (␢)
<p> All text should go between beginning and ending paragraph tags </p>

<center> center your TEXT or IMAGE </center>

<b> make it BOLD </b>

SIZE OF TEXT
< font size=1 > font size 1 < /font > < font size=2 > font size 2 < /font > < font size=3 > font size 3 < /font > ...
< font size=7 > font size 7 < /font >
COLOR OF TEXT
< font color=green > text < /font >
< font color=#31007b > text < /font >

DIFFERENT FONTS
< font face="Verdana" > Verdana Font < /font >
< font face="Arial" > Arial Font < /font >
< font face="Arial Black" > Arial Black Font < /font >
< font face="Helvectica" > Helvectica Font < /font >
< font face="Impact" > Impact Font < /font >
< font face="Comic Sans MS" > Comic Sans MS < /font >
< font face="Georgia" > Georgia Font < /font >
< font face="Times New Roman" > Times New Roman Font < /font >
< font face="Courier New" > Courier New Font < /font >
< font face="Trebuchet MS" > Trebuchet MS Font < /font >

<font size=5 color=green face="Trebuchet MS">
You can put in as many of these elements that you want in the FONT tag like I did in the above FONT tag to get this style, size, and color of the text you see here.
</font>
