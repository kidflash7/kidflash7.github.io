"Lance LINK Detective Chimp":
Now you want to make a page that consists of several HTM files, so we need to create links to the other HTM files that are part of your big page. Links are the topics you've been clicking on in the left frame in this tutorial. You click on them and they bring up a new HTM file in the right frame. You can make anything on your page a link: images, text, you name it. First I'll show you how to make a part of your text a link. Notice that the link is underlined, this is automatic with text, that way it'll stand out as a link even if the link is the same color as the surrounding text. I made it blue so it'll stand out as a link when you click it, it becomes yellow for an instant and then when you come back to the page it is red. This is done in the body tag with link, alink, vlink. Ok I'll show you how it is done in the HTML that follows:
< a href="url_address" > the text you want clickable < /a >
The "url_address" is where the new page is located and the name of the page. A local link (link to the same web site) is specified with a relative URL (without http://www....). My new page is in the same directory so I just called the name of the HTM file I want to come up when they click on the link. But suppose I want to call on a page "colors.htm" that is in a folder called "files" I would access it by the following local "url_address" "files/colors.htm". Now let's make an image a link...
< a href="new_page.htm" >
< img src="image_name.jpg" >
< /a >
