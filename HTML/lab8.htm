Now suppose you have a really long page, we've all encountered these pages. It is very tedious to keep hitting that scroll bar on the side of the window. A better way to handle this is to make it so the user clicks on a menu and it jumps down to the part of the page they want to view. I don't use this much because it's not very practical to have pages that are too big.
